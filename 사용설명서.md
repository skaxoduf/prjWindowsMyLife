## 🔧 EXE 파일 생성하기

### 방법 1: 자동 빌드 (추천)

1. `build_exe.bat` 파일을 더블클릭
2. 자동으로 PyInstaller 설치 및 EXE 생성
3. 완료!

### 방법 2: 수동 빌드

```bash
# 1. PyInstaller 설치
pip install pyinstaller

# 2. EXE 생성
pyinstaller --onefile --console --name "BrowserLauncher" browser_launcher.py
```

## ⚙️ URL 설정하기

### 1단계: config.ini 파일 열기

생성된 `dist` 폴더 또는 exe와 같은 위치에 `config.ini` 파일을 복사합니다.

### 2단계: URL 수정

`config.ini` 파일을 메모장으로 열어 원하는 URL로 수정합니다:

```ini
[Edge]
url1 = https://www.naver.com
url2 = https://www.daum.net

[Chrome]
url1 = https://www.google.com
url2 = https://www.youtube.com
url3 = https://www.github.com
url4 = https://www.stackoverflow.com
url5 = https://www.reddit.com

[Brave]
url1 = https://www.youtube.com
```

### 설정 설명:

- **각 브라우저마다 url1, url2, url3... 형식으로 무제한 추가 가능!**
- **[Edge]** 섹션: Edge 브라우저에서 열 사이트들 (예시: 2개)
- **[Chrome]** 섹션: Chrome 브라우저에서 열 사이트들 (예시: 5개, 모두 탭으로 열림)
- **[Brave]** 섹션: Brave 브라우저에서 열 사이트들 (예시: 1개)
- **개수 제한 없음**: 1개부터 100개 이상까지 자유롭게!

## ▶️ 실행하기

### 폴더 구조

```
📁 원하는 폴더
 ├── BrowserLauncher.exe  ← 실행 파일
 └── config.ini            ← 설정 파일 (반드시 같은 폴더에!)
```

### 실행 방법

1. `BrowserLauncher.exe` 파일을 더블클릭
2. 자동으로 모든 브라우저가 실행됩니다!
3. 3초 후 프로그램이 자동 종료됩니다

### 바로가기 만들기

1. `BrowserLauncher.exe` 우클릭 → 바로가기 만들기
2. 바로가기를 바탕화면이나 시작 메뉴에 배치
3. 이름 변경: "브라우저 일괄 실행" 등

## 🎨 커스터마이징

### URL 개수 추가/제거

**각 브라우저별로 자유롭게 URL 추가/제거 가능 (코드 수정 불필요!):**

```ini
[Edge]
url1 = https://www.naver.com
url2 = https://www.daum.net
url3 = https://www.google.com
url4 = https://www.yahoo.com
# ... 원하는 만큼!

[Chrome]
url1 = https://www.google.com
url2 = https://www.youtube.com
# ... 100개까지도 가능!
```

**중요:** url 번호는 1부터 순서대로 (url1, url2, url3...) 작성해야 합니다. 중간 번호를 건너뛰면 그 이후 URL은 무시됩니다.

### 브라우저 비활성화하기

특정 브라우저를 사용하지 않으려면 `config.ini`에서 해당 섹션을 삭제하거나 주석 처리:

```ini
# [Brave]
# url = https://www.youtube.com
```

## 🔍 문제 해결

### ❌ "config.ini 파일을 찾을 수 없습니다"

**원인:** config.ini 파일이 exe와 같은 폴더에 없습니다.

**해결방법:**
- config.ini 파일을 BrowserLauncher.exe와 같은 폴더에 복사

### ❌ 특정 브라우저가 실행되지 않음

**원인:** 브라우저가 설치되지 않았거나 시스템 PATH에 없습니다.

**해결방법:**

**Brave 브라우저:**
- 실행 명령어가 다를 수 있습니다
- `browser_launcher.py`에서 `'brave'`를 다음 중 하나로 변경:
  - `'brave-browser'`
  - 또는 전체 경로: `'C:\\Program Files\\BraveSoftware\\Brave-Browser\\Application\\brave.exe'`

**Chrome 브라우저:**
- 기본 설치 경로에 없는 경우 전체 경로 사용:
  - `'C:\\Program Files\\Google\\Chrome\\Application\\chrome.exe'`

### ❌ Chrome에서 탭이 따로따로 열림

**원인:** Chrome 버전 또는 설정 문제

**해결방법:**
Python 코드 수정:
```python
# 한 번에 실행하는 대신
for url in chrome_urls:
    subprocess.Popen(['start', 'chrome', url], shell=True)
    time.sleep(0.3)
```

### 💡 콘솔 창 숨기기

창이 나타나지 않게 하려면:

```bash
pyinstaller --onefile --noconsole --name "BrowserLauncher" browser_launcher.py
```

단, 오류 메시지를 볼 수 없으므로 정상 작동 확인 후 사용하세요.

## 📝 추가 기능 아이디어

### 1. 시작 프로그램 등록

Windows 시작 시 자동 실행:
- `Win + R` → `shell:startup`
- 바로가기를 해당 폴더에 복사

### 2. 딜레이 추가

브라우저 사이에 대기 시간 추가:
```ini
[Settings]
delay = 2  # 초 단위
```

### 3. 프로필 기능

여러 config 파일 사용:
- `config_work.ini` - 업무용
- `config_personal.ini` - 개인용
